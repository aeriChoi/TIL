### 자바스크립트는 동적 타입 언어이며 7가지 내장 타입을 가지고 있다.

### 01. 데이터 타입

* 변수에 저장하는 데이터의 종류\(값, **value** \)를 뜻한다.

```javascript
let a = 1;
let b = "hi";
let c = true;
let d = false;
```

| **Variables** | **Values** |
| :--- | :--- |
| a | **1** |
| b | **"hi"** |
| c | **true** |
| d | **false** |
